### 1. LocaionFuncSet
GPS provider를 이용하여 안드로이드 서비스의 상태변경 어플리케이션.
저장된 GPS값의 영역으로 사용자의 디바이스가 이동할 경우 그 저장한 서비스 동작으로 디바이스의 상태를 변경함.
서비스 동작여부는 SharedPreferences를 이용하여 저장
WI-FI, BlueTooth, 소리 및 진동상태를 변경함.

##### 참여한 부분: 개인프로젝트
